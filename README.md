# CrashGuard – Smart Accident Detection & Evidence Saver

## Overview

CrashGuard is a smartphone-based accident detection and emergency response system designed to improve road safety. The application automatically detects potential vehicle crashes using smartphone sensors, records evidence, tracks GPS location, and alerts emergency contacts when an accident occurs.

The goal is to provide an affordable alternative to dedicated dashcams while ensuring quick emergency response and reliable accident evidence collection.

---

## Problem Statement

Road accidents often result in:

* Lack of reliable evidence
* Delayed emergency assistance
* Difficulty sharing accurate location information
* High costs associated with dedicated dashcam systems

CrashGuard addresses these issues using only a smartphone and cloud services.

---

## Features

### Accident Detection

* Detects sudden impacts using smartphone motion sensors
* Monitors abnormal movement patterns

### Emergency Alert System

* 10-second countdown to cancel false alarms
* Automatically sends alerts to emergency contacts

### GPS Tracking

* Captures real-time accident location
* Shares coordinates with emergency contacts

### Evidence Recording

* Records video evidence using the smartphone camera
* Stores evidence for future reference

### Cloud Storage

* Uploads recorded accident footage to the cloud
* Prevents loss of critical evidence

---

## Working Flow

1. User starts driving.
2. CrashGuard monitors sensor data.
3. Accident is detected.
4. 10-second cancellation countdown begins.
5. If not cancelled:

   * Emergency contacts are notified.
   * GPS location is shared.
   * Video evidence is saved.
   * Evidence is uploaded to cloud storage.
